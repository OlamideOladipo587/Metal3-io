About Us

Metal3-io pronounced: Metal Kubed is an open-source project that simplifies the deployment and management of bare metal infrastructure using Kubernetes. The framework uses the Kubernetes API server to manage the lifecycle of bare metal hosts, enabling users to manage both virtual machines and bare metal hosts using a unified interface.

Our Mission

The Metal3-io project mission is to provide a Kubernetes-native way of managing bare metal infrastructure, making it as easy as managing virtualized infrastructure.

Our Community

The Metal3-io project is maintained by a diverse community of contributors from various organizations and is part of the Open Infrastructure Foundation. Help build the future of Metal3.io
